Assetra Redesign Checklist

Assetra 目前需要修改的 UI 項目

根據最新畫面評估，整理成可執行的改版清單。目標是把目前的金融 Dashboard 從「乾淨的 SaaS 介面」提升成「高級資產管理平台 / Personal Wealth Operating System」。

Priority

最高優先順序

1. 提高精準對比

目前整體太灰，Card、背景、NavRail 層級不夠明確。

2. 強化數字字體層級

金融產品最重要的是數字權威感，Hero number 要更有份量。

3. 建立品牌級 Design System

定義顏色、字體、卡片、圖表與 active state，而不是逐頁微調。

Contrast

1. 整體對比需要提高

目前畫面乾淨，但 Card、背景、Rail 的灰階太接近，導致整體有點糊。高級感不是低對比，而是「精準對比」。

區域建議顏色用途
App Background#F5F7FA整體背景，提供乾淨但不刺眼的底色。
Card Background#FFFFFF所有財務資料卡片，保持資訊清晰。
NavRail Background#FBFCFD左側導航，比主背景稍亮，與內容區分開。
Card Borderrgba(15, 23, 42, 0.08)比目前更清楚，但不要變成重框線。

建議

  • 讓背景、卡片、NavRail 有明確層級。
  • 用 border + subtle shadow 分區。
  • 保留金融產品的 calm 感。

避免

  • 所有區域都用接近的淡灰色。
  • 靠大量粗框線分區。
  • 把畫面做得太白、太刺眼。
Typography

2. Typography 需要更有金融權威感

目前字體太平均，導致大數字沒有足夠的重量。資產管理產品的核心是數字，數字必須有「可信、穩重、專業」的感覺。

文字類型建議尺寸字重備註
Hero Number48–56px700–800例如淨資產 NT$7,135,403。
Card Number24–32px700例如投資資產、負債比率、投資淨損益。
Section Title14–16px700清楚但不要搶走數字焦點。
Metadata12–13px500日期、同步狀態、輔助說明。
Hero Number 可使用 letter-spacing: -0.03em，讓大數字更緊實、更有高級金融產品感。
NavRail

3. NavRail 需要精品化

NavRail 的資訊架構很好，但視覺上還有一些 WPF / 後台系統感。需要讓它更像高級金融工具的主導航。

項目目前問題建議修改
Item Height偏低，像一般清單。調整為 44–48px。
Active State還是偏 selected list item。使用 3px 左側 active bar + 淡藍背景。
Icon存在感偏弱。統一線條風格，例如 Lucide / Phosphor。
Section Label層級不夠精緻。小字、低對比、加字距，例如 ANALYTICS / ASSETS。
Color System

4. 建立金融產品色彩系統

目前色彩偏 generic SaaS。資產管理產品可以加入更沉穩的金融色彩，讓品牌更有記憶點。

Primary
#1E3A8A
主品牌色，適合 active state、重點按鈕、重要文字。
Positive
#059669
正向數值，例如成長、收益、完成。
Negative
#DC2626
負向數值，例如虧損、風險、警示。
Premium Accent
#C8A96B
低彩度金色，可用於高級標籤、Premium 功能、財富目標。
Chart

5. Chart 需要從工程圖改成財富趨勢圖

現在圖表已經比前版好，但仍偏工程 / BI dashboard。財富趨勢圖應該更輕、更乾淨、更像金融產品。

  • 線條建議 2px，不要太硬。
  • Grid 透明度降低約 70%。
  • Area fill 使用極淡透明度，不要整片灰。
  • Hover tooltip 顯示日期、淨值、日變化。
  • Y 軸資訊降低存在感，避免干擾主趨勢。
FIRE

6. FIRE 區塊需要更像核心差異化功能

FIRE 是 Assetra 的差異化功能之一，但目前看起來像 placeholder。建議把它設計成一個真正的財務自由模擬入口。

目前建議
立即計算開始退休模擬
尚未計算 FIRE 目標預測你的財務自由時間
單純 CTA顯示預估自由日期、進度、投入金額、成功率。
Status Bar

7. Status Bar 要更 Wealth-aware

Status Bar 的方向是對的，適合桌面專業工具。但文字要從 technical status 轉成財務語意。

  • 同步完成 · 3 個帳戶已更新
  • 市場已收盤 · 今日淨值 +0.07%
  • 資料更新於 2026-05-21 08:03
  • 離線模式 · 使用本機資料
Final Checklist

最小可執行改版清單

  1. 調整背景、卡片、NavRail 的顏色層級。
  2. 放大 Hero Number，建立數字字體規格。
  3. 修改 NavRail active state：左側 3px indicator + 淡藍背景。
  4. 統一 icon style 與 icon size。
  5. 降低 chart grid 與 area fill 的存在感。
  6. 重新設計 FIRE 卡片文字與 CTA。
  7. 把 Status Bar 文字改成財務語意。
  8. 整理成 Assetra Design System tokens。
現階段不建議繼續增加功能。優先把視覺系統定下來，Assetra 的產品感會提升非常明顯。
