# 資產管理 CIS — 企業識別系統

## 品牌方向

### 品牌定位
一個現代資產管理品牌，核心聚焦於：

- 長期價值
- 穩定性
- 機構級信任
- 精準
- 結構化智慧
- 低調奢華
- 現代金融科技

- Apple 等級的極簡主義

- Bloomberg 式的資訊清晰度
- 現代 SaaS 產品的一致性
